# 📊 AI & Data Science Job Market Analysis (2020-2026)

Este proyecto es una aplicación interactiva desarrollada para el curso **Talento Tech - Nivel Integrador (Intermedio)**. La plataforma permite explorar las tendencias, salarios y roles emergentes en el campo de la Inteligencia Artificial y la Ciencia de Datos.

## 🚀 Características de la Aplicación

* **Landing Page Profesional:** Una interfaz de bienvenida con contexto claro sobre el dataset.
* **Visualización de Datos:** Gráficos estadísticos avanzados utilizando `Seaborn` y `Matplotlib`.
* **Análisis Temporal:** Evolución de salarios y demanda de roles desde 2020 con proyecciones al 2026.
* **Documentación Integrada:** Sección de pestañas (Tabs) con el diccionario de datos y metodología aplicada.
* **Filtros Dinámicos:** Capacidad de segmentar la información por año y otras variables clave.

## 📋 Requisitos e Instalación

1.  **Clonar el repositorio:**
    ```bash
    git clone [https://github.com/tu-usuario/nombre-del-repo.git](https://github.com/tu-usuario/nombre-del-repo.git)
    ```

2.  **Instalar dependencias:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Configuración de Kaggle:**
    Asegúrate de tener tus credenciales de Kaggle configuradas para que `kagglehub` pueda descargar el dataset automáticamente.

4.  **Ejecutar la aplicación:**
    ```bash
    streamlit run app.py
    ```
